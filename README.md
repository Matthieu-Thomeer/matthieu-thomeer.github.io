# Site de recherche — Matthieu Thomeer

Site personnel présentant mes sujets de recherche, hébergé via GitHub Pages.

## Structure des fichiers

```
├── index.html        ← Page principale du site
├── README.md         ← Ce fichier
└── assets/           ← Images et fichiers statiques (optionnel)
```

## Comment modifier le site

Ouvrez `index.html` dans un éditeur de texte et cherchez les sections marquées avec des crochets `[...]` pour les remplacer par vos vraies informations :

- `[votre domaine de recherche]` → votre domaine
- Cartes de recherche (Thème principal, Axe 02, Axe 03) → vos vrais sujets
- Section Publications → vos vraies publications
- Section À propos → votre vraie biographie
- `votre@email.fr` → votre adresse email réelle
